# React Template

내가 쓸려고 만든 템플릿임.

## 🚀 사용 방법

### degit 사용 (추천)
```bash
npx degit kingcat47/React_default-template .
npm install
```

- ⚡ Vite + React + TypeScript
- 🎨 SCSS Modules
- 🧩 재사용 가능한 UI 컴포넌트
- 📁 체계적인 폴더 구조
- 🔧 ESLint 설정
- 📱 반응형 디자인
